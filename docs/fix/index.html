fix

fix

Define a reusable pattern → replacement rule across your entire codebase.

fix is Delta's most versatile construct. Write a pattern with $VAR capture groups and a replacement — Delta applies it across every file matching the scope glob. Chain multiple fixes in one file and apply them all at once.

Syntax

fix fixName {
  pattern:  { old pattern with $VAR }
  replace:  { new pattern with $VAR }
  scope:    "glob/**/*.ts"
  severity: bug | perf | style | security
  note:     "explanation"  // optional
}

Example

fix strictNull {
  pattern:  { $X == null }
  replace:  { $X === null }
  scope:    "**/*.ts"
  severity: bug
  note:     "Loose equality matches both null and undefined"
}

apply fix strictNull to project preview

Fields

FieldRequiredDescription
name✓Unique identifier — referenced by apply
pattern✓Code pattern to match, with optional $VAR capture groups
replace✓Replacement code — $VAR references are substituted
scope✓Glob pattern: files to scan (e.g. src/**/*.ts)
severity✓bug | perf | style | security — shown in output
note✓Explanation shown in diff output

Tips

  • Always test with preview first: apply fix myFix to project preview
  • $VAR captures any expression in that position
  • Multiple fixes can be defined and applied in the same file
  • Scope globs follow fast-glob syntax — ** crosses directory boundaries
