Getting Started

Install Delta and run your first code transformation in under 5 minutes.

Installation

Delta requires Node.js 18 or higher.

npm install -g @delta-lang/cli

Verify the installation:

delta --version

Your first transform

Create a file called transform.delta:

fix strictNull {
  pattern: { == null }
  replace: { === null }
  scope:   "**/*.ts"
  severity: bug
  note:    "use strict equality"
}

apply fix strictNull to project preview

Preview what will change:

delta run transform.delta --preview

Apply the changes:

delta run transform.delta

CLI Commands

CommandDescription
delta run <file>Run a .delta file
delta run <file> --previewPreview changes without writing
delta check <file>Parse and validate without running
delta initScaffold a starter transform.delta
