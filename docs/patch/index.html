patch

Surgically replace a function or block in a specific file.

Use patch when you know exactly which function or code block needs to change in a specific file. Delta uses the TypeScript Compiler API or tree-sitter to locate the target by AST — not text search.

Syntax

patch "path/to/file" in Language {
  find fn functionName() {}
  replace with { newBody }
  why: "reason"  // optional
}

Example

patch "src/auth/login.ts" in TypeScript {
  find fn hashPassword(password: string) {}
  replace with {
    return await bcrypt.hash(password, 12);
  }
  why: "upgrade to async bcrypt"
}

Fields

FieldRequiredDescription
file✓Relative path to the file to patch
in Language✓Target language: TypeScript, JavaScript, Python, Go, Rust, Java
find fn name()✓Locate a function by name
find block label✓Locate a labeled code block
replace with { }✓New implementation — replaces the entire function body
why✓Human-readable reason, stored in commit message and diff output

Tips

  • patch only touches one function in one file — use fix for multi-file patterns
  • The file path is relative to where you run delta from
  • Use preview to see the diff before applying: delta run file.delta --preview
  • The replace body replaces the function body, not the signature
