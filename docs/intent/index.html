intent

Inject code at every function entry or exit point across matching files.

intent lets you add logging, tracing, metrics, guards, or any other cross-cutting concern to every function in scope — without manually editing each file. Delta finds every function declaration and injects your code at the specified hook point.

Syntax

intent "description" {
  scope:    "glob/**"
  language: Language
  inject:   { codeToInject }
  where:    function_entry | function_exit
  preserve: "comment to keep"  // optional
  guard:    guardName          // optional
}

Example

intent "add request tracing" {
  scope:    "src/api/**"
  language: TypeScript
  inject:   { tracer.startSpan(arguments.callee.name); }
  where:    function_entry
}

intent "log all exits" {
  scope:    "src/api/**"
  language: TypeScript
  inject:   { logger.debug("exit"); }
  where:    function_exit
}

Fields

FieldRequiredDescription
label✓Description of the intent
scope✓Glob: which files to modify
language✓Target language for the injected code
inject✓Code block to inject
where✓function_entry or function_exit — where to inject
preserve✓Comment pattern to preserve existing injections
guard✓Guard name — intent is only applied if guard passes

Tips

  • intent is the fastest way to add observability to an entire service
  • Combine with guard to only inject when certain conditions hold
  • Use function_exit to add cleanup, flush, or finalisation code
  • intent + trace is a powerful combination for security auditing
