Six constructs. Infinite transforms.

Every transformation maps to one of six composable keywords.

patch

Patch

Surgically replace a specific function or block in a specific file. Uses the TypeScript Compiler API or tree-sitter for AST-accurate targeting.

fix

Fix

Define a reusable pattern → replacement rule and apply it across your entire codebase with glob scoping and $VAR capture groups.

intent

Intent

Inject code at every function entry or exit point without manually touching each file. Add logging, tracing, or guards everywhere at once.

migrate

Migrate

Upgrade from one API to another. Rename symbols, move files, replace patterns — all in one declarative migration block.

guard

Guard

Assert invariants before applying any transform. Block apply if a function is missing, a forbidden import exists, or raw SQL is detected.

trace

Trace

Follow data through your codebase. Track where req.body.password flows, flag paths that reach console.log or unparameterised queries.
